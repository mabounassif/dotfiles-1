dotfiles
========

#VIM

In order to use the vim configuration please install Vundle:

```shell
$ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

Then while in vim use: ```:PluginInstall```

To use YouCompleteMe:
```shell
$ cd ~/.vim/bundle/YouCompleteMe && ./install.sh
```

#ZSH
